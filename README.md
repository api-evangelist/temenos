# Temenos

APIs for Temenos banking and financial services platform, providing cloud-native, cloud-agnostic, API-first banking solutions including core banking, digital banking, payments, wealth management, financial crime mitigation, and more.

**Website:** [https://www.temenos.com](https://www.temenos.com)
**Developer Portal:** [https://developer.temenos.com](https://developer.temenos.com)

**Tags:** Banking, Cloud Banking, Core Banking, Digital Banking, Financial Services, Fintech, Open Banking, Payments, Wealth Management

**Modified:** 2026-05-03

Core banking system API for managing accounts, transactions, and customer data. Covers enterprise, holdings, meta, order, party, product, reference, settings, and system API categories with over 290 endpoints spanning retail, corporate, wealth, treasury, and Islamic banking.
